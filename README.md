🐍 Snake Game
프로젝트 소개
이 프로젝트는 HTML, CSS, JavaScript만을 사용하여 구현한 간단한 스네이크 게임입니다.
별도의 라이브러리나 프레임워크 없이 하나의 HTML 파일 안에 모든 코드가 포함되어 있어, 브라우저에서 바로 실행할 수 있습니다.

주요 기능
방향키(↑, ↓, ←, →)로 뱀을 조작

랜덤 위치에 생성되는 음식 먹기

음식 섭취 시 뱀의 길이 증가 및 점수 상승

벽이나 자기 몸에 부딪히면 게임 오버

점수 표시 기능

실행 방법
저장소를 클론하거나 snake.html 파일을 다운로드합니다.

bash
git clone https://github.com/사용자명/저장소명.git
snake.html 파일을 브라우저에서 실행합니다.

Mac에서는 TextEdit로 작성 후 .html 확장자로 저장하면 됩니다.

저장 시 순수 텍스트(Plain Text) 형식으로 저장해야 합니다.

브라우저에서 파일을 열면 게임이 시작됩니다.

코드 구조
HTML: 게임 캔버스와 점수 표시 영역 정의

CSS: 배경, 캔버스 스타일, 점수판 디자인

JavaScript: 게임 로직 (뱀 이동, 음식 생성, 충돌 판정, 점수 관리)

파일 구성
코드
snake.html   # 게임 전체 코드가 포함된 단일 파일
게임 화면
검은색 배경의 캔버스 위에서 녹색 뱀이 움직이며 빨간색 음식이 등장합니다.

상단에 현재 점수가 표시됩니다.

향후 개선 아이디어
난이도 조절 (속도 증가)

모바일 터치 조작 지원

배경 음악 및 효과음 추가

최고 점수 저장 기능(LocalStorage 활용)
